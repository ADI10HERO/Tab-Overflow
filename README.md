# Tab-Overflow


## Basic Features:   
- Login and SignUp   
- Add Activities   
- Add Tasks inside activities   
- Add URLs inside Tasks  
- Search, Rename, Delete Activities  
- Search, Rename, Delete Tasks 
- Display Tabs Opened Currently in the Browser (Real-Time)
